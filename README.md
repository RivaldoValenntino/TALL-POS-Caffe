## About Application

Point of Sales Application built with  TALL STACK (Tailwind,AlpineJS,Laravel,Livewire) and integrated with Midtrans Payment Gateaway

## Installation Guide

Open project and run this command

```
$ composer install 
```
also run
```
$ npm install
```
and
```
$ php artisan:migrate --seed
```

- Require PHP ^8.2
- Node JS v21.5.0
- NPM v10.2.4
